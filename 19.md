## Code Guide: Doormat Pattern Program
------------------------------
## Step 1: Program Start and Welcome Banner

* Heading: Welcome Banner Initialization
* Code:

if __name__ == '__main__':
    print('\n','*'*30,"Welcome to the Program",'*'*30,'\n')

* Explanation:
The first line ensures this script runs directly. The print statement multiplies the * symbol 30 times to create a decorative banner. The \n tags create clean, empty lines before and after the text.
* Output:

 ****************************** Welcome to the Program ****************************** 


------------------------------
## Step 2: The Input Validation Loop

* Heading: Loop for Input Verification
* Code:

    while True:
        N = int(input("Enter Odd Positive Natural Number: "))
        print()

        if N%2 == 0 and N<0:
            print(f"{N} is a Negative Even Number! Please try again.\n")
        elif N%2 == 0: 
            print(f"{N} is an Even Number! Please try again.\n")
        elif N<0:
            print(f"{N} is a Negative Number! Please try again.\n") 
        elif N==0:
            print(f"{N} is Zero! Natural Numbers should be greater than Zero.\n")    
        else:
            break

* Explanation:
This loop runs forever until it hits a break. It asks the user for a number. It checks for four bad scenarios like negative numbers, even numbers, or zero. If the number fails a check, it prints a specific error and loops again. If the number is a valid positive odd number, it hits the else block and exits the loop. (For the output below, let's assume the user enters a valid number like 7).
* Output:

Enter Odd Positive Natural Number: 7



------------------------------
## Step 3: Setting the Canvas Width

* Heading: Canvas Width Calculation
* Code:

    M = N*3

* Explanation:
This line calculates the total width of the doormat design. It multiplies your chosen number N by 3 and saves it in a variable called M. If N is 7, then M becomes 21 characters wide.
* Output:

 [1] 

------------------------------
## Step 4: Printing the Top Triangle

* Heading: Upper Pattern Generation
* Code:

    for i in range(1, N, 2):
        print((i*".|.").center(M, "-"))

* Explanation:
This loop draws the upper half of the pattern. The range starts at 1 and counts up to N, skipping even numbers. In each row, it multiplies the design .|. by the number i. Then, .center(M, "-") centers that pattern and fills the remaining left and right sides with dashes.
* Output:

--------.|.--------
------.|..|..|.------
----.|..|..|..|..|.----


------------------------------
## Step 5: Printing the Center Welcome Message

* Heading: Center Message Alignment
* Code:

    print("Welcome".center(M, "-"))

* Explanation:
This line prints the exact center of the doormat. It takes the word "Welcome" and uses the same .center() tool to place it right in the middle, filling the edges with dashes up to the total width M.
* Output:

-------Welcome-------


------------------------------
## Step 6: Printing the Bottom Triangle

* Heading: Lower Pattern Generation
* Code:

    for i in range(N-2, -1, -2):
        print((i*".|.").center(M, "-"))

* Explanation:
This loop flips the top pattern upside down to finish the doormat. The range starts just below your maximum number and counts backward by 2 down to 1. Just like before, it multiplies the .|. pattern and centers it with dashes.
* Output:

----.|..|..|..|..|.----
------.|..|..|.------
--------.|.--------


------------------------------
## Step 7: Program End and Goodbye Banner

* Heading: Exit Banner Initialization
* Code:

    print('\n','*'*30,"Thank You",'*'*30,'\n')

* Explanation:
This final line prints a closing message to tell the user the program has successfully finished. It uses the same 30-star decoration format as the opening welcome banner.
* Output:

 ****************************** Thank You ****************************** 

[2]
