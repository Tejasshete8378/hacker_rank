### 3) Heading: Generating and Printing Combinations

**Code:**

```python
    print("Result is as Follows: ")
    for i in range(1, k+1):
        final = combinations(new_s, i)
        for j in final:
            result = "".join(j)
            print(result)

```

**Explanation:**

1. We loop through a range from `1` to `k` (inclusive) to get combinations of varying lengths.
2. `combinations(new_s, i)` generates all combinations of length `i`.
3. The nested loop iterates through each combination (`j`), and `"".join(j)` converts the tuple of characters back into a clean string.
4. Each resultant string is printed on a new line.

**Output:**

```text
Result is as Follows: 
A
C
H
K
AC
AH
AK
CH
CK
HK

```
