
```text
chars = 'abcdefghijklmnopqrstuvwxyz'
```

#### Code:
```python
# Inside the loop for i = 0
s = "-".join(chars[size - 1 : i : -1] + chars[i:size])
```

We start a loop with `i = 0`.

### Step 3: Loop For Second Row (i = 1)
#### Code:
```python
# Inside the loop for i = 1
s = "-".join(chars[size - 1 : i : -1] + chars[i:size])
lines.append(s.center(4 * size - 3, "-"))
```
#### Explanation:
Now the loop moves to `i = 1`.
* `chars[2:1:-1]` gives us `'c'`.
* `chars[1:3]` gives us `'bc'`.
* Combining them gives `'cbc'`.
* Adding dashes gives `'c-b-c'`.
* This string is 5 characters long. We use `.center(9, "-")` to add two dashes on the left and two dashes on the right to make it 9 characters long.
#### Output:
```text
s = 'c-b-c'
lines = ['c-b-a-b-c', '--c-b-c--']
```

### Step 4: Loop For Third Row (i = 2)
#### Code:
```python
# Inside the loop for i = 2
s = "-".join(chars[size - 1 : i : -1] + chars[i:size])
lines.append(s.center(4 * size - 3, "-"))
```
#### Explanation:
The loop moves to the last index, `i = 2`.
* `chars[2:2:-1]` gives an empty string `''`.
* `chars[2:3]` gives us `'c'`.
* Combining them gives `'c'`.
* Adding dashes leaves it as `'c'`.
* This string is 1 character long. We center it to 9 characters by adding four dashes on each side.
#### Output:
```text
s = 'c'
lines = ['c-b-a-b-c', '--c-b-c--', '----c----']
```

### Step 5: Mirror and Join Rows
#### Code:
```python
rangoli = "\n".join(lines[::-1] + lines[1:])
print(rangoli)
```
#### Explanation:
We flip the `lines` list to build the top half first, then add the bottom half.
* `lines[::-1]` flips the list to `['----c----', '--c-b-c--', 'c-b-a-b-c']`.
* `lines[1:]` skips the first item of the original list to get the bottom rows: `['--c-b-c--', '----c----']`.
* We join all these rows together with a new line character `\n` and print the final shape.
#### Output:
```text
----c----
--c-b-c--
c-b-a-b-c
--c-b-c--
----c----
```

# Why 4*size - 3 is taken:

The math formula 4 * size - 3 is used because it calculates the exact total width of the longest middle row in your pattern.
Here is the breakdown of why that math works, using your example where size = 3.
## 1. Count the Letters
The middle row uses letters going backward and forward. [1] 

* The number of letters is always: 2 * size - 1
* For size 3: 2 * 3 - 1 = 5 letters (c, b, a, b, c)

## 2. Count the Dashes
Every letter has a dash between it. The number of dashes is always one less than the number of letters.

* The number of dashes is: (2 * size - 1) - 1
* This simplifies to: 2 * size - 2
* For size 3: 2 * 3 - 2 = 4 dashes

## 3. Add Them Together
To find the total width, you add the letters and the dashes together:

* (2 * size - 1) + (2 * size - 2)
* This simplifies perfectly to: 4 * size - 3
* For size 3: 4 * 3 - 3 = 9 total characters wide

## Visual Example (size = 3)
If you count every character in the middle row, you get exactly 9:

c - b - a - b - c
1 2 3 4 5 6 7 8 9

By making every row exactly 4 * size - 3 characters wide, the center() tool knows how many extra dashes to add to the smaller rows so everything lines up.
