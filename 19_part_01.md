# Step 2

## Heading: Number Verification for N = 7
## Code:
```python
    while True:
        N = int(input("Enter Odd Positive Natural Number: "))
        print()
        
        if N%2 == 0 and N<0:
            print(f"{N} is a Negative Even Number! Please try again.\n")
        elif N%2 == 0: 
            print(f"{N} is an Even Number! Please try again.\n")
        elif N<0:
            print(f"{N} is a Negative Number! Please try again.\n") 
        elif N==0:
            print(f"{N} is Zero! Natural Numbers should be greater than Zero.\n")    
        else:
            break
```
## Explanation:
The loop asks you to type a number. We enter the number `7`. The code checks the conditions: `7` is not even, not negative, and not zero. It matches the `else` block and calls `break` to exit the loop safely.

## Output:
```text
Enter Odd Positive Natural Number: 7

```

# Step 3

## Heading: Calculating Grid Width
## Code:
```python
    M = N*3
```
## Explanation:
The variable `M` sets the total width of the design. Since our number `N` is `7`, the code multiplies `7` by `3`. This means `M` is now equal to `21`. Every line of the pattern will be exactly 21 characters wide.

## Output:
```text
(No text is printed. In computer memory, M = 21)
```

# Step 4

## Heading: Creating Top Pattern Rows
## Code:
```python
    for i in range(1, N, 2):
        print((i*".|.").center(M, "-"))
```
## Explanation:
The `range(1, 7, 2)` gives three numbers: `1`, `3`, and `5`. 
* For `i = 1`: It prints 1 copy of `.|.` centered in 21 spaces using dashes.
* For `i = 3`: It prints 3 copies of `.|.` (`.|..|..|.`) centered in 21 spaces.
* For `i = 5`: It prints 5 copies of `.|.` centered in 21 spaces.

## Output:
```text
---------.|.---------
------.|..|..|.------
---.|..|..|..|..|.---
```

# Step 6

## Heading: Creating Bottom Pattern Rows
## Code:
```python
    for i in range(N-2, -1, -2):
        print((i*".|.").center(M, "-"))
```
## Explanation:
This loop creates the lower half of the design. The `range(5, -1, -2)` gives three numbers in reverse order: `5`, `3`, and `1`.
* For `i = 5`: It prints 5 copies of `.|.` centered in 21 spaces.
* For `i = 3`: It prints 3 copies of `.|.` centered in 21 spaces.
* For `i = 1`: It prints 1 copy of `.|.` centered in 21 spaces.

## Output:
```text
---.|..|..|..|..|.---
------.|..|..|.------
---------.|.---------
```
